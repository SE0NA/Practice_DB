#### 03. forward action
- forward action은 include action과 사용법은 유사하지만 요청 페이지를 다른 페이지로 전환할 때 사용
- 브라우저 URL 창에는 최초 요청 페이지가 표시 되기 때문에 처리 페이지 정보를 숨기거나 MVC 패턴의 컨트롤러와 같이 특정 기능 수행 후 다른 페이지로 이동해야 하는 경우 유용하게 사용 가능
